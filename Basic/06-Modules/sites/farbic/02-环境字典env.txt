env是 Python 字典的子类，既用作设置，也用于任务间数据空间共享。
环境字典 fabric.state.env 是作为全局的单例实现的，为方便使用也包含在 fabric.api 中。 env 中的键通常也被称为“环境变量”。



运行环境即设置
常用内容：
env.hosts
env.user        在建立 SSH 连接时默认使用本地用户名
env.password    显式设置默认连接或者在需要的时候提供 sudo 密码
env. warn_only  布尔值，用来设置 Fabric 是否在检测到远程错误时退出



settings 会话管理器



环境即状态共享



其他考虑
env支持属性访问进行读写操作
格式化的时候更简短 print("Executing on %(host)s as %(user)s" % env) 而不是  % (env.host, env.user)



环境变量完整列表
大部分都可以直接操作，但最好还是使用 context_managers ，可以通过 settings 或特定的上下文管理器，如 cd 。
